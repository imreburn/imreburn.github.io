If we encode a complete binary tree to an array, a parent node and two child nodes have a certain relationship in their indices. 
With that, we can easily navigate binary tree up and down. 
It seems just right, however, I cannot come up with a simple proof.
Moreover, if I googled, articles in page 1 do not include the proof.
Maybe it is too simple to write the proof.
But I want the proof!

*Prove:* When a complete or full binary tree $$T$$ is represented by an array and
a parent node $$p$$ has the index $$i$$, then
the index of the left and the right child node is $$2i + 1$$ and $$2i + 2$$, respectively.
